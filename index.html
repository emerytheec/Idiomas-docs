Idiomas

Sistema de localizacion automatica para mundos de VRChat — Compatible con PC y Quest

Idiomas es un sistema de localizacion para mundos de VRChat que traduce la interfaz de usuario (Canvas) a 11 idiomas de forma automatica. Detecta el idioma del jugador, traduce los textos y permite cambiar de idioma en tiempo real con un dropdown.

Incluye herramientas de editor para escanear tu escena, generar claves de traduccion, y traducir automaticamente usando APIs gratuitas (MyMemory y Lingva) sin necesidad de registro ni pago. Tambien puedes exportar a CSV para trabajar con traductores humanos.

Funciona con UdonSharp y BehaviourSyncMode.Manual. Diseñado para que localizar tu mundo sea tan facil como escanear, hacer clic, y traducir.

Que puedes hacer con Idiomas?

  • Traduccion automatica de Canvas — escanea tus Canvas y traduce todos los textos a 11 idiomas con unos pocos clics
  • Deteccion automatica de idioma — detecta el idioma del jugador via API de VRChat, zona horaria del sistema, o idioma fallback
  • Auto-traduccion con APIs gratuitas — usa MyMemory y Lingva para traducir sin registro ni pago (5000 caracteres/dia)
  • 11 idiomas soportados — ingles, español, japones, coreano, chino simplificado, chino tradicional, ruso, portugues, frances, aleman y catalan
  • Fuentes CJK integradas — japones, coreano y chino funcionan sin fuentes adicionales gracias a Noto Sans de VRChat
  • Exportar / Importar CSV — comparte traducciones con traductores humanos via Google Sheets o Excel
  • Textos dinamicos — parametros {0}, {1}, {2}, prefijos, sufijos y formato rich text
  • Pluralizacion — soporte para variantes _zero, _one, _other
  • Editor multiidioma — toda la interfaz del editor esta traducida a los 11 idiomas
  • Selector de idioma — prefab con dropdown TMP listo para usar en tu mundo

Componentes del Sistema

Componentes Runtime

LocalizationManager
Componente central que gestiona las traducciones, detecta el idioma y notifica a todos los localizadores.
CanvasLocalizer
Localiza todos los textos de un Canvas completo con un solo componente.
TextLocalizer
Localiza un texto individual con parametros dinamicos, prefijos y rich text.

Herramientas del Editor

Auto-Traduccion
Ventana para traducir automaticamente textos faltantes usando MyMemory y Lingva.
CSV Export / Import
Exporta e importa traducciones en formato CSV para colaborar con traductores humanos.
