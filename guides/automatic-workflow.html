Flujo Automatico

Localiza todos tus Canvas con unos pocos clics usando la Configuracion Rapida

Cuando usar el flujo automatico

  • Cuando casi todos tus Canvas estan escritos en el mismo idioma
  • Cuando quieres que el sistema haga la mayor parte del trabajo
  • Cuando no necesitas controlar cada texto individualmente
  • Es el flujo recomendado para la mayoria de mundos

Paso 1: Colocar el prefab

Arrastra el prefab LocalizationManager.prefab a la escena de tu mundo. Lo encuentras en: Packages/Idiomas/Prefabs/LocalizationManager.prefab

El prefab incluye un Canvas con un dropdown de seleccion de idioma ya configurado. Si no necesitas el dropdown, puedes eliminar el Canvas hijo y dejar solo el componente LocalizationManager.

Paso 2: Configurar el inspector

1 2 3 4 5
1 Idioma del Editor — cambia el idioma de la interfaz del inspector. No afecta a las traducciones del juego, solo a los textos del editor. 2 Archivo de traduccion — aqui se asignara automaticamente el JSON con las traducciones. Al principio esta vacio; se creara con el primer escaneo. 3 Idioma Fallback — selecciona el idioma que se usara cuando no se pueda detectar el del jugador. Elige el idioma en el que estan escritos la mayoria de tus Canvas. 4 Escanear Escena — busca todos los Canvas de la escena. Este paso solo detecta, no modifica nada. 5 Estadisticas — informacion sobre el estado actual de la localizacion (idiomas, claves, Canvas).
ⓘ El idioma fallback es importante: es el idioma "de emergencia". Si un jugador tiene un idioma que no esta en tu archivo de traduccion, vera los textos en el idioma fallback.

Paso 3: Escanear la escena

Al pulsar "Escanear Escena", el sistema busca todos los Canvas en la escena y los muestra en una lista:

1 2 3 4 5
1 Lista de Canvas encontrados — cada fila muestra un Canvas con su nombre y ruta en la jerarquia. 2 Idioma base global — selecciona el idioma en que estan escritos tus textos. Se aplicara a TODOS los Canvas en la Configuracion Rapida. 3 Configuracion Rapida: Localizar Todo — el boton principal que automatiza todo el proceso. 4 Añadir CanvasLocalizer individual — si prefieres, puedes localizar Canvas de forma individual. 5 Estadisticas de textos — muestra cuantos textos TMP y Legacy tiene cada Canvas.

Paso 4: Configuracion Rapida

Este es el paso clave. Al pulsar "Configuracion Rapida: Localizar Todo", el sistema ejecuta automaticamente:

  1. Añade un componente CanvasLocalizer a cada Canvas detectado en la escena
  2. Genera un ID unico para cada Canvas basado en su nombre en la jerarquia
  3. Escanea todos los textos hijos de cada Canvas (TextMeshProUGUI y Legacy Text)
  4. Evalua cuales necesitan traduccion — excluye automaticamente numeros puros ("123", "45.6"), signos ("-", "/", ":"), y textos vacios
  5. Genera claves de traduccion basadas en la jerarquia (ej: "settings_panel_volume_label")
  6. Crea o actualiza el archivo JSON con todas las claves y el texto del idioma base seleccionado
  7. Registra todos los CanvasLocalizer con el LocalizationManager
ⓘ Si el archivo de traduccion no existia, se crea automaticamente en la carpeta Idiomas_Data/ dentro de Assets/. Si ya existia, las claves nuevas se agregan sin borrar las existentes.

Paso 5: Revisar resultados

Despues de la Configuracion Rapida, puedes seleccionar cualquier Canvas para ver su CanvasLocalizer y revisar los textos detectados:

1 2 3 4 5
1 Canvas ID — identificador unico generado automaticamente. 2 Idioma base — el idioma que seleccionaste en la Configuracion Rapida. 3 Checkbox de inclusion — puedes desmarcar textos que no quieras traducir. 4 Tabla de textos — cada texto con su clave, contenido actual, ruta y estado. 5 Exportar al JSON y Aplicar — si haces cambios manuales, pulsa aqui para guardarlos.
ⓘ En la mayoria de casos, despues de la Configuracion Rapida no necesitas tocar nada aqui. Solo revisa si quieres excluir algun texto especifico o ajustar alguna clave.

Paso 6: Traducir

Con las claves en el JSON, es momento de traducir. Ve a la seccion "Traduccion" del LocalizationManager o usa el menu Tools > Idiomas > Auto-Traducir.

El proceso resumido:

  1. Pulsa "Auto-Traducir Idiomas Faltantes"
  2. Selecciona los idiomas que quieres traducir
  3. Pulsa "Traducir X textos con MyMemory"
  4. Revisa las traducciones en la vista previa
  5. Pulsa "Guardar al JSON"

Sigue la guia completa de Auto-Traduccion para el paso a paso detallado.

Paso 7: Probar

Entra en Play Mode en Unity. El sistema detecta tu idioma automaticamente. Usa el dropdown del prefab para probar diferentes idiomas y verificar que todo se traduce correctamente.

⚠ Los caracteres japoneses, coreanos y chinos pueden verse como cuadrados (□) en el editor de Unity. Esto es normal — en VRChat se renderizan correctamente gracias a las fuentes Noto Sans incluidas en el cliente.

Resultado

Despues de estos 7 pasos, tu mundo tiene:

  • Todos los Canvas localizados automaticamente
  • Un archivo JSON con traducciones en los idiomas que elegiste
  • Deteccion automatica del idioma del jugador
  • Un dropdown para que el jugador cambie idioma manualmente
  • Todo funcional tanto en el editor como en VRChat
