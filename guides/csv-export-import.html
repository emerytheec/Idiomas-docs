Exportar / Importar CSV

Comparte traducciones con traductores humanos via Google Sheets o Excel

Descripcion

Si necesitas traducciones de mayor calidad o en idiomas que las APIs automaticas no manejan bien, puedes exportar tu archivo JSON a CSV, compartirlo con traductores humanos, y reimportar las traducciones de vuelta.

Abrir la ventana

Menu: Tools > Idiomas > Exportar-Importar CSV

Formato del CSV

key,en,es,ja,ko,zh-CN
btn_start,Start,Inicio,スタート,시작,开始
btn_close,Close,Cerrar,閉じる,닫기,关闭
settings_title,Settings,Configuracion,設定,설정,设置
  • Primera columna: clave de traduccion
  • Columnas siguientes: un idioma por columna
  • Primera fila: cabecera con los codigos de idioma
  • Celdas vacias: traducciones pendientes

Exportar a CSV

  1. Abre la ventana desde Tools > Idiomas > Exportar-Importar CSV
  2. Selecciona tu archivo JSON de traducciones
  3. Pulsa "Exportar a CSV"
  4. Elige donde guardar el archivo .csv
  5. Abre el CSV en Google Sheets o Excel
  6. Comparte con tus traductores
ⓘ Google Sheets es ideal para colaborar. Sube el CSV, comparte el enlace con tus traductores, y cuando terminen descarga el archivo actualizado como CSV.

Importar desde CSV

  1. Abre la ventana de Exportar-Importar CSV
  2. Selecciona tu archivo JSON de destino
  3. Pulsa "Importar desde CSV"
  4. Selecciona el archivo .csv actualizado
  5. Las traducciones nuevas se fusionan con las existentes
⚠ La importacion NO sobreescribe traducciones existentes. Solo añade traducciones nuevas (celdas que estaban vacias). Si quieres reemplazar una traduccion existente, editala directamente en el JSON.

Consejos para traductores

Si compartes el CSV con traductores, indicales estas reglas:

  • No modificar la primera fila (cabecera de idiomas)
  • No modificar la primera columna (claves de traduccion)
  • Dejar vacias las celdas que no quieran traducir
  • Respetar las etiquetas de rich text (<color>, <b>, etc.)
  • Respetar los placeholders {0}, {1}, {2} — son variables dinamicas que se rellenan en el juego
  • Tener en cuenta la longitud del texto — algunos idiomas necesitan mas espacio que otros

Flujo de trabajo colaborativo

  1. Tu: Exportas el JSON a CSV
  2. Tu: Subes el CSV a Google Sheets y compartes el enlace
  3. Traductores: Rellenan las celdas vacias con las traducciones
  4. Tu: Descargas el Sheets como CSV
  5. Tu: Importas el CSV de vuelta al JSON
  6. Tu: Verificas en Play Mode que todo se ve correctamente
